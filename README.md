# Smart Waste-to-Energy System 🌱⚡

An IoT-based smart city solution that converts waste into renewable energy, powers public Wi-Fi, and improves air quality.

## Features
- Smart bins with sensors
- Waste-to-energy conversion
- Public Wi-Fi support
- Air purification

## Tech Used
- Python
- IoT (ESP32)
- HTML/CSS (planned)
