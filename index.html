🌱 Smart Waste to Energy System

This project converts waste into energy and powers smart city features like Wi-Fi.

About the Project

This project focuses on solving urban waste problems by converting organic waste into renewable energy. It also supports smart city development by powering public Wi-Fi and improving air quality.

Key Features

  • Smart IoT-based waste bins for efficient waste tracking
  • Conversion of organic waste into renewable biogas energy
  • Public Wi-Fi powered using generated energy
  • Air purification using algae-based coating
  • Reward system to encourage proper waste disposal
